# Kanban Board (React + Vite)

Мини-проект для портфолио: Kanban-доска с колонками `Backlog / In Progress / Done`.

## Реализованный функционал

- drag-and-drop карточек между колонками
- добавление задач с выбором приоритета (`High / Medium / Low`)
- редактирование заголовка и описания карточки
- удаление карточек
- быстрый перенос карточки кнопками `← / →`
- поиск по задачам (по заголовку и описанию)
- фильтр задач по приоритету
- очистка колонки `Done` одной кнопкой
- сохранение состояния в `localStorage`

## Стек

- React 18
- Vite
- CSS3
- localStorage API
- HTML5 Drag and Drop API

## Запуск локально

```bash
npm install
npm run dev
```

После запуска открой адрес из консоли (обычно `http://localhost:5173`).

## Сборка

```bash
npm run build
npm run preview
```
